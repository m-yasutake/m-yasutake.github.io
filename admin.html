Map Data 🗺️

Upload GPX routes, CSV point files, or add points from Google Maps. Changes appear on the Map page immediately.

📄 Upload GPX Route(s)

Upload .gpx files to add bike routes to the map. Each file is stored in Firebase Storage and registered in Firestore.

📂

Drag & drop GPX files or click to browse

📋 Upload CSV Point(s)

Upload a .csv file with columns: name, latitude, longitude, and optionally url and Type (Onsen, Campsite, Roadside Station, Must See, Hotel, Other).

📂

Drag & drop CSV files or click to browse

📍 Add Google Maps Point

Paste a Google Maps URL (full or short link) to add a single point directly to Firestore.
